Exercício 4

- Calculadora com switch

4- Calculadora com switch:

Enunciado: Crie uma calculadora em PHP que receba duas entradas numéricas e uma operação (soma, subtração, multiplicação ou divisão) através de um formulário HTML. Dependendo da operação escolhida, o sistema deve realizar a operação e exibir o resultado.

-------------------------------------

Passos a seguir:

Crie um formulário HTML com dois campos de entrada para os números e uma opção para selecionar a operação.

O PHP deve usar switch case para realizar a operação e exibir o resultado no navegador.

--------------------------------------
